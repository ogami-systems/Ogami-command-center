# Ogami Command Center

This repository is the operating brain for Ogami, Platinum Bill Pay experiments, and OpenClaw development.

## Mission

Build reusable operating systems that make businesses simpler, clearer, and more effective.

Nothing in this repository should be a one-off.

Everything should become a reusable asset.

## Repository Structure

01-rules
Operating rules, standards, routing, security, and model selection.

02-agents
AI employees with specific jobs.

03-workflows
Repeatable processes.

04-clients
Client operating manuals.

05-prompts
Reusable prompts.

06-memory
Persistent context and long-term knowledge.

07-docs
Architecture, ideas, and documentation.
